ClassicAltTab

Have the classic Windows NT Alt+Tab style in modern versions of Windows.

GitHub Repository:

https://github.com/kruct/classicalttab

How to install:

First, download a ZIP file containing the latest version of ClassicAltTab.

Once you have downloaded the ZIP file, extract it to a folder of your choice.

Open that folder and run the Installer.cmd file and wait for the installation to finish.
